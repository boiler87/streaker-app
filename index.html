// SYSTEM OVERVIEW

STREAKER v2.0 is an advanced behavior modification terminal designed to gamify discipline through negative reinforcement and denial. This system tracks "streaks" of time, calculates experience points (XP), and assigns ranks based on endurance.

1. DASHBOARD & HUD

  • STATUS: Indicates if a session is currently active.
  • SESSION: Real-time counter of current streak duration.
  • RANK: Your current clearance level based on XP.
  • GOAL: Visual progress bar towards your set objective (Days or Date).

2. ACTIVITY LOGGING

  • LOG ACTIVITY: Start a new streak. Previous active streaks must be ended first.
  • EDIT/DELETE: Modify timestamps or purge records via the history list.
  • END & RESTART: Terminates current session, calculates final XP, and immediately begins a new timer.

3. JOURNAL & CHECK-INS

  • STATUS CHECK: Daily boolean tracking for Locked state, Release (failure), or Denial (success).
  • LOGS: Text entries for tracking mental state. Can be Pinned for visibility.
  • PRIVACY: Journal text is NEVER displayed on public profiles.

4. ORACLE SYSTEM

Consult the Oracle on the Dashboard for permission guidance. Results are RNG-based but skewed towards denial.
